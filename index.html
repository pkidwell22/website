Margot v3.0

A model-selectable AI assistant for macOS and iOS with 84 Google Cloud tools, scheduled automation, and memory that persists.

Choose your coordinator from any model on OpenRouter, with ten of the latest models already configured. Orchestrate tasks across Google Cloud, Drive, Gmail, & local files, automate browser tasks, and create scheduled workflows, backed by four-layer cognitive memory and context laundering that keeps every conversation sharp and affordable.

Capabilities

Google Cloud

84 tools across Analytics 4, Search Console, Merchant Center, Gmail, Drive, Sheets, Calendar, and PageSpeed Insights. Multi-account OAuth with automatic routing by connection type.

Research

Three tiers: standard chat (5–15s), light research with subtopic pipelines (30–90s), and deep research with 4 independent agents and cross-pollination (90–180s).

Desktop & Browser

21 desktop tools with three-level shell safety. Browser automation via Playwright with 48 skills, deterministic recovery, and release gating.

Media Generation

Image generation via Gemini 3 Pro (up to 4K), video via Veo 3.1 and Sora 2, voice synthesis via ElevenLabs with streaming.

Scheduled Automation

Run full coordinator workflows unattended on cron, interval, or slot schedules. Verification contracts, mutation guardrails, and cooperative cancellation for safe background execution.

Context Laundering

Large tool payloads are processed before LLM reinjection—raw, summarized, or truncated based on content type. Integrated across chat, research, and scheduled task runtimes.

Generated Skills

Create new tools from natural language. The skill generator produces Python, validates via AST analysis and automated tests, then hot-registers into the coordinator’s tool set—no restart required.

Architecture

A model-selectable coordinator runs a ReAct loop (up to 30 iterations) and delegates to specialized subagents. Route through any model on OpenRouter—with automatic resilience that retries and falls back on pre-first-chunk failures.
